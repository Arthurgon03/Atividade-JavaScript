If, else, else if: é usado para executar um bloco de código se uma condição for verdadeira. 
O else é usado para executar um bloco de código se a condição do if for falsa. Já o else if é utilizado quando 
você tem várias condições a serem testadas em sequência. Cada else if é verificado se as condições anteriores 
forem falsas e a condição atual for verdadeira.

Switch Case: o switch em JavaScript executa diferentes ações com base no valor de uma expressão. Cada case 
representa um valor possível da expressão. Após a correspondência, o bloco de código associado é executado,
e o break interrompe a execução. Se nenhum caso corresponder, o bloco de código do default é executado. 
É uma forma organizada de lidar com múltiplas condições.

For Loop: é uma estrutura de controle que executa um bloco de código repetidamente, usando uma inicialização 
para definir uma variável de controle, uma condição para determinar a continuação do loop e um incremento para 
atualizar a variável de controle a cada iteração. É ideal para situações em que você sabe quantas vezes precisa 
repetir uma ação.

While: ele repete um bloco de código enquanto uma condição é verdadeira. Não possui inicialização nem incremento, 
sendo ideal para situações em que o número de iterações não é conhecido. É útil quando se precisa repetir uma
 ação até que uma condição específica seja atendida.
